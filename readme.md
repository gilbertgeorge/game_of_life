# Based on [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)

## Demo
https://gilbertgeorge.github.io/game_of_life/

## How to run
### Console version
```
cargo run
```

### Web version
```
wasm-pack build
cd www
npm install
npm start
```
Open http://localhost:8080 in your browser
